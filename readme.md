# 👩‍💻 Ingrid Solís
## Data, Docencia y Transformación Digital

Este repositorio reúne material de trabajo, relatorías y proyectos aplicados en el ámbito de análisis de datos, visualización y ciencia de datos.
Mi enfoque es práctico, orientado a negocio y siempre con una mirada clara hacia la toma de decisiones.

---

## 🎯 ¿Qué encontrarás aquí?

Este espacio está organizado en tres líneas principales:

### 📚 Teaching / Relatorías

Material y presentaciones utilizadas en cursos y capacitaciones:
- Python para análisis de datos
- Visualización con Power BI y Jupyter Notebooks
- Fundamentos de ciencia de datos

El objetivo es explicar temas complejos de manera simple, práctica y aplicable.

### 📊 Proyectos aplicados

Casos prácticos desarrollados con datos reales o simulados:
- Análisis exploratorio de datos
- Visualización geoespacial con GeoPandas
- Construcción de indicadores
- Ejemplos orientados a toma de decisiones

Aquí los datos se conectan con preguntas reales de negocio.

### 🧪 Experimentos y exploración

Espacios de prueba para aprender y descubrir:
- Nuevas librerías
- Modelos de machine learning
- Pruebas de visualización
- Ideas en desarrollo

No todo está totalmente concluido, pero cada experimento aporta al aprendizaje.

---

## 🛠️ Tecnologías utilizadas

- Python (Pandas, NumPy, GeoPandas, Matplotlib)
- SQL
- Power BI
- Jupyter Notebooks
